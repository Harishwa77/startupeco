# **App Name**: EchelonAI

## Core Features:

- AI Mode Selector: An intuitive user interface to select between 'founder', 'investor', 'intern', and 'evolution' modes for interacting with the AI Engine.
- Dynamic Input Form: Generates mode-specific input fields and forms, adhering to the AI Engine's INPUT STRUCTURE for comprehensive data entry.
- Engine API Interface: Seamless integration with the Advanced AI Startup Ecosystem Engine's backend API to submit requests and retrieve detailed analytical responses.
- Structured AI Output Viewer: Presents complex JSON responses from the AI Engine in a human-readable, interactive, and clearly structured format.
- AI Startup Idea Tool: An LLM-powered tool to help users refine or generate initial startupIdea and other textual descriptions based on high-level inputs for 'founder' mode.
- Performance Score Visualizations: Graphical representations of the 0-100 numerical scores (e.g., marketViability, riskScore) to provide quick, insightful data analysis.

## Style Guidelines:

- A dark, sophisticated background (#15191F) chosen to evoke a professional, analytical, and data-rich environment suitable for strategic insights.
- Primary action color: A deep, analytical blue (#396DBF), providing a strong contrast and signifying clarity, strategy, and technological focus against the dark background.
- Accent color: A bright, clear cyan (#5FCAE6), strategically used for highlighting data points, interactive elements, and critical information to guide user attention effectively.
- Headline font: 'Space Grotesk' (sans-serif) to provide a modern, tech-forward, and bold feel, reflecting innovation and analytical precision.
- Body text font: 'Inter' (sans-serif) for exceptional readability of analytical content, detailed reports, and quantitative data, ensuring clarity and neutrality.
- Code font: 'Source Code Pro' (monospace) specifically for displaying structured JSON outputs, guaranteeing clear and uniform code presentation.
- Utilize minimalist line icons throughout the interface, aligning with a data-driven, analytical aesthetic and ensuring a clean, uncluttered visual experience.
- A clean, dashboard-like layout with distinct, well-organized sections for input controls and AI engine outputs, optimized for efficient data consumption and user workflow.
- Subtle and purposeful animations for state changes, loading indicators, and score transitions, designed to enhance user experience without introducing distractions or perceived latency.
